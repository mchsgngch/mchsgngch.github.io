Web Designer Michisuga Noguchi

埼玉県在住のWebデザイナー、野口　理清(のぐち　みちすが)です。オンラインスクールでWeb制作について学び、小売業界から憧れていたWebデザイナーへと転職活動中です。これからさらにWeb制作の技術や知識を身につけて、運営も含めユーザビリティの考慮したWebサイトを作成できるデザイナーになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。JavaScriptなどを使用した動きのあるデザインやマルチデバイスにも対応したサイト作りやも現在勉強中です。

Communication

コミュニケーション能力が自信があります。前職のスポーツ専門店で年間個人売上で全店1位になりました。クライアントの意向に汲みとったサイト作りに活かしていきます。

Management

前職において3年間管理職を務めました。売上管理から設備、従業員の管理をしてきました。新店舗の立ち上げ経験もあります。プロジェクトの進捗管理に活かしていきます。
